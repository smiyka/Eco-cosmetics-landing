# About:
Here is a fully responsive landing page, created for:
> The design 1440px
> Desktop 1280px
> Tablet 640px
> Mobile (> 320px)

# Stack:
> HTML5
> CSS3 | Sass (SCSS)
> BEM methodology
> Flex | Grid

# How to start
> Fork the repo.
> Clone the forked one.
> Run npm install (or just npm i).
> Run npm start.
